ngautoComplete 操作示例 参数列表

市面上这么多自动补全插件，那我为什么要用ng autocomplete？
  • 功能全面，细节处理好，无bug。
  • 按键支持，ESC，Enter, TAB
  • 支持大数据列表，可滚动列表
  • 键盘按上下按键的时右侧滚动条自动滚动
  • 可设置按键触发列表延迟时间
  • 支持选中时onSelect回调函数
  • 可设置获得文本框焦点自动打开
  • 可设置是否区分大小写
  • 可设置输入框占位文字
  • 形状上：可设置下拉列表的高度，宽度，以及显示多少项数
  • 样式上：表格各行变色，hover变色，显示正在加载中图标提醒
  • 细节上：鼠标移动到列表上显示完全名称，防止名称太长显示不完全
  • 细节上：下拉列表项不能被鼠标拉黑
  • 支持ajax远程操作
  • 可设置ajax接口查询参数名称修改
  • 可设置ajax请求方法
  • 可指定ajax返回对象自定义指定字段为数据

操作示例

var data = ['111','222',...,'999'] 测试数据总数：2000

例 1：本地数据基本使用;placehold占位符

new autoComplete({
    el: '#t1',
    data: data,
    placeholder:'试一下↑↓，Tab键，Enter键吧！'
});

例 2：获得焦点自动打开;选择后回调

new autoComplete({
    el: '#t2',
    data: data,
    autoOpen:true,
    placeholder:'点击我自动打开',
    onSelect: function (value) {
        alert('您选择的是：'+value);
    }
});

例 3：按键延迟，2秒后打开列表框

new autoComplete({
    el: '#t3',
    data: data,
    placeholder:'延迟2秒后打开',
    keyupDelay:2000
});

例 4：默认下拉项数：5个；并且设置列表宽度为200px

new autoComplete({
    el: '#t4',
    data: data,
    placeholder:'下拉项数：5个',
    max:5,
    width:200
});

例 5：不区分大小写,如输入AA；延迟关闭2秒

new autoComplete({
    el: '#t5',
    data: data,
    placeholder:'不区分大小写,延迟关闭2秒',
    closeDelay:2000,
    matchCase:false
});

例 6：指定data对象的数组字段为数据

var dataobj = {ddd:data};
new autoComplete({
    el: '#t6',
    data: dataobj,
    placeholder:'指定data对象的数组字段data.ddd',
    root:'ddd'
});

例 7：ajax; url为cgi接口； paramName为查询参数名； root为指定data对象的数组字段为数据

new autoComplete({
    el: '#t7',
    url:'test.php',
    placeholder:'ajax测试,请求参数名设置',
    paramName:'word',
    root:'data'
});

参数说明

名称 可选值 描述
el require jquery Selector 要自动提示的inputbox Element对象
data [] 数据使用本地对象数组
placeholder string 输入框默认提示文字
autoOpen {false | true} 输入框获得焦点自动打开
onSelect function(value){} 选中的时候回调函数，;value为选择的项
ajax
url string ajax远程cgi地址
paramName 'q' : string ajax远程请求的参数名称
method {GET | POST} string ajax请求方法
root string 指定data对象的数组字段为数据
附加
listHeight 180 : int 下拉框高度(px)
max int 下拉框显示的项目个数
width int 下拉框的宽度(px)
matchCase {true | false} 是否区分大小写
keyupDelay 200 : int 按键触发时间(ms)
closeDelay 300 : int 延时关闭时间(ms)
